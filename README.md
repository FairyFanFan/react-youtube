## Environment

Get a Youtube Data API key

Create a `.env.local` file

```
source .env.local
```

write this in .env.local

```
REACT_APP_YT_API_KEY=AIzaxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## Installing dependencies


```
npm install
```
or 

```
yarn install
```
